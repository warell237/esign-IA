# ESIGN AI — Design System

## Couleurs
- bg-esign-dark      → fond de page
- bg-esign-darker    → sidebar, header
- bg-esign-blue      → cards, panels
- bg-esign-electric  → boutons, liens actifs
- text-esign-lightblue → textes secondaires
- text-esign-accent  → badges, alertes
- text-esign-success → messages de succès
- text-esign-error   → messages d'erreur

## Animations
- animate-fade-in    → messages chat
- animate-slide-up   → modales
- animate-pulse-glow → bouton IA actif
